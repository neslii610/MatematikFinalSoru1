Matematik Final – 1. Soru Açıklaması: Bu program, ikinci dereceden denklemlerin köklerini bulmak için hazırlanmıştır. Kullanıcıdan alınan a, b ve c katsayıları ile önce diskriminant değerini hesaplar. Bu değere göre denklemin reel kökleri varsa hesaplayıp ekrana yazdırır. Kodlama aşamasında C++ dili kullanılmıştır. Programın çalışma mantığını gösteren ekran çıktısı da görsel olarak projeye eklenmiştir.
